# somascan-host-proteins-sarscov2

Code and workflows for the study “Host proteins associated with strong neutralizing SARS-CoV-2 antibody responses in a South African cohort.” Includes R scripts for SomaScan® proteomics analysis, identifying host proteins, pathways, and predictors linked to neutralizing antibody responses.
